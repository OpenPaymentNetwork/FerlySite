This directory contains message templates for a WingCash site. You can change
them and upload your changes as a ZIP file. You should include HASHES.json
in the uploaded ZIP file so WingCash can accurately detect which files you
changed. You should not change HASHES.json.

If you want to revert a template to the default, delete everything from the
file (leaving a zero byte file) and upload it. WingCash will restore the
default template and the default template will be in the next ZIP download.
